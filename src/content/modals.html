Full Screen Modal

A specialized fixed position modal that expands to fit the browser window.

In mobile:

It becomes an overlay with width and height equal to the viewport width and height. Modal-header content is hidden and modal-footer content becomes the heading.

Change modal button text into icons by combining class modal-btn-icon with an icon class name. For example, add modal-btn-icon and icon-ok on the primary button to change the text into a checkmark in mobile.

In mobile safari (iOS 8.3), any content inside an iframe that triggers a browser reflow will cause the iframe to scroll to the top.
