--- title: Collapsible Search section: Components (JS) ---

A jQuery plugin that opens and closes a search bar in mobile.

Data Attributes:

Enable by data attribute with data-toggle="collapsible-search" and specify at what screen width to run collapsible-search by adding data-breakpoint="991", default is 768.

Javascript:

Enable by javascript with $('#myElement').collapsibleSearch(); and specify at what screen width to run collapsible-search with:

$('#myElement').collapsibleSearch({
    breakpoint: 991 // default 768
});

Changing the breakpoint requires changing the media queries for basic search in the navbar component.
