List Group Item Flex

Align content inside .list-group-item with flexbox with .list-group-item-flex.

Use a combination of .autofit-col and .autofit-col.autofit-col-expand to control the size of each cell.

show-quick-actions-on-hover will need Javascript to add class table-focus on the table row when an item inside is focused to make it keyboard accessible.
