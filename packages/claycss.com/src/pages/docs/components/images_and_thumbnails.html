Image Aspect Ratios

Sometimes you can't control the size of an image, define a specific size for your images with aspect-ratios. The base class {literal}aspect-ratio{/literal} maintains a 1 to 1 ratio relative to its container's width, add class {literal}aspect-ratio-vertical{/literal} to maintain a 1 to 1 ratio relative to its container's height.

Use {literal}aspect-ratio-3-to-2{/literal}, {literal}aspect-ratio-4-to-3{/literal}, {literal}aspect-ratio-8-to-5{/literal}, or {literal}aspect-ratio-16-to-9{/literal} to maintain the specific ratio relative to its container or create your own by setting {literal}padding-bottom{/literal} to the ratio you want e.g. 2 to 1 {literal}.aspect-ratio-2-to-1 { padding-bottom: 50% }{/literal}.

Horizontal 1:1 Aspect Ratio with Varying Image Sizes

Images inside {literal}aspect-ratio{/literal} are aligned bottom by default. Use class {literal}aspect-ratio-top{/literal} or {literal}aspect-ratio-middle{/literal} to align top or vertically center, respectively.

Vertical 1:1 Aspect Ratio with Varying Image Sizes

Images inside {literal}aspect-ratio-vertical{/literal} are aligned left by default. Use class {literal}aspect-ratio-right{/literal} or {literal}aspect-ratio-center{/literal} to align right or horizontally center, respectively.

Crop Img

Use the component {literal}crop-img{/literal} to crop an image based on its container size, the container's position must be set to relative, absolute, or fixed, or set width and height on {literal}crop-img{/literal} directly.

Horizontal Crop Img Alignment

Horizontally align the image inside crop-img with classes {literal}crop-img-center{/literal} and {literal}crop-img-right{/literal}, default alignment is to the left.
