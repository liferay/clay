Clay Dual Listbox

The size attribute on each .clay-reorder select[multiple] should be the maximum number of options that the inputs can contain. This ensures that clay-dual-listbox will not grow and shrink when items are moved from one input to another.

If shifting content isn't an issue, the size attribute can be updated to match as options get moved around.

Clay Dual Listbox

Add the class clay-reorder-footer-invisible to clay-reorder to hide the contents of the footer. It uses visibility: hidden; to ensure the boxes do not shift.
