	Check the [Lexicon](https://liferay.design/lexicon) [List

</div>

List sections help separate content by a specific category or type.

## Example

Align content inside a `.list-group-item` element with a flexbox with `.list-group-item-flex`.

Use a combination of `.flex-col` and `.flex-col.flex-col-expand` to control the size of each cell.

Add the class `show-dropdown-action-on-active` to display `dropdown-menu`s when active in List Groups.

<div className="alert alert-warning">
	Direct descendants of <code>flex-col</code> become block level elements by
	default. See{' '}
	<a href="https://www.w3.org/TR/css-flexbox-1/#flex-items">
		https://www.w3.org/TR/css-flexbox-1/#flex-items
	</a>
	. If you want to display content using <code>floats</code>, <code>
		inline
	</code>, or <code>inline-block</code>, wrap the content with a block level element
	such as a div.
</div>
