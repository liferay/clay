Nested Nav

Nested navigation, also collapsible (requires Bootstrap Collapse Plugin). Add class nav-nested to the outermost nav to indent each nested nav.

Set the icon for the closed state by setting the variable $nav-collapse-icon-closed in _variables.scss or overwrite in your css by setting .nav-collapse-icon:before { content: "" }

Set the icon for the open state by setting the variable $nav-collapse-icon-open in _variables.scss or overwrite in your css by setting the content property of .collapsed .nav-collapse-icon:before { content: "" }
