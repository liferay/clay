--- title: Sheet section: Components ---

Sheet

Sheets should be used as containers for Forms. There should only be one sheet-header and sheet-footer per sheet. Multiple sections inside sheets can be spaced using sheet-section.

Sheet Title

Sheet text should be used for any kind of explanatory text. The Sheet Title and Sheet Text are contained inside a sheet-header.

Sheet Subtitle

Sheet Tertiary Title

Sheet Subtitle and Sheet Tertiary Title are contained inside a sheet-section.

Sheet

Use .sheet-lg with .sheet to set a max-width of 800px or ceil(map-get($container-max-width, lg) * 0.83333) on the sheet. This is used as containers for forms.
