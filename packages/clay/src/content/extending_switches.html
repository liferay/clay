Custom Switch using Images

Customize the button style in the off position with input.switch:empty ~ .switch-bar:after {}.

Customize the switch background style in the off position with input.switch:empty ~ .switch-bar:before {}.

Customize the icon in the off position with input.switch:empty ~ .switch-bar .switch-icon-off {}.

Customize the button style in the on position with input.switch:checked ~ .switch-bar:after {}.

Customize the switch background style in the on position with input.switch:checked ~ .switch-bar:before {}.

Customize the icon in the on position with input.switch:checked ~ .switch-bar .switch-icon-on {}.
