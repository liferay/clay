Toolbar

Quickly prototype a toolbar for your portlet, control panel, or site with this component. Vertically align content by adjusting the height of .toolbar-group, .toolbar-group-right, and .toolbar-group-expand.

Toolbar shouldn't be used to create a toolbar for your inline/text editing application; use btn-toolbar for that.

Fill Remaining Space with Text

This pattern is specifically for use with long text that should be displayed in a single line and the remainder truncated. Truncated text should have a title attribute or a tooltip so a user still can access the full content.

Use .toolbar-group-expand-text with .toolbar-group and .toolbar-group-right to fill the remaining space with text in your toolbar. .toolbar-group-expand-text must be the last toolbar-group declared. It can be vertically aligned by matching the css property line-height to the height of its sibling toolbar-groups.

Vertically centering different size content.

Variable height icons/content can be vertically centered by placing each one inside a .toolbar-group-content. Variable height toolbar-groups are vertically aligned by matching the height of each toolbar-group to the tallest one.
