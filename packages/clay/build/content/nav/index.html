Nav

Basic Nav

Minimally styled navigation with .nav.

Use .nav-heading on a list element with a nested <span></span> or <a></a> to style the heading.

  • Basic Information
  • Details
  • Catagorization
  • Documents and Media
  • Site Template
<ul class="nav">
    <li class="nav-heading"><span>Basic Information</span></li>
    <li><a href="#">Details</a></li>
    <li><a href="#">Catagorization</a></li>
    <li><a href="#">Documents and Media</a></li>
    <li><a href="#">Site Template</a></li>
</ul>

Nested Basic Nav

Nested navigation, also collapsible (requires javascript). Add class nav-nested to the outermost nav to indent each nested nav.

Set the icon for the closed state by setting the variable $nav-collapse-icon-closed in _variables.scss or overwrite in your css by setting .nav-collapse-icon:before { content: "" }

Set the icon for the open state by setting the variable $nav-collapse-icon-closed in _variables.scss or overwrite in your css by setting the content property of .open a > .nav-collapse-icon:before { content: "" }
