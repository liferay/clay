Nav

Use the base nav class to create a minimally styled navigation.

  • Basic Information
  • Details
  • Catagorization
  • Documents and Media
  • Site Template
<ul class="nav">
    <li class="nav-heading"><span>Basic Information</span></li>
    <li><a href="#">Details</a></li>
    <li><a href="#">Catagorization</a></li>
    <li><a href="#">Documents and Media</a></li>
    <li><a href="#">Site Template</a></li>
</ul>

Nested navigation, also collapsible (requires javascript). Add class nav-indent to the outermost nav to indent each nested nav.
