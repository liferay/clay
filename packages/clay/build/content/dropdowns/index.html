Scrolling Content

Add class inline-scroller to any element and set the height or width to make it scroll inline! The default height is 125px, but can be overridden to any height/width.

Dropdown wide / full

Add class dropdown-wide to the parent of a dropdown-menu to make the dropdown-menu BIG. The default width is 500px, but can be overridden to any width. The dropdown-full class can be used to make the dropdown as wide as the navigation, see dropdown examples.
