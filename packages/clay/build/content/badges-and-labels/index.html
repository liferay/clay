Badges, Labels and Stickers

Badges

Badges help highlight important information such as notifications or new and unread messages. Badges have circular borders by default, for rounded borders, See labels.

Fixed Width and Height Badges

Use badge-monospaced to create a badge that has a fixed width and height. It is useful for showing the number of new messages or pending requests in your application.

Stickers

Stickers are absolutely positioned monospaced badges, place them anywhere relative to your container.

133 133 133 133 133 133 
<span class="sticker sticker-default">133</span>
<span class="sticker sticker-danger">133</span>
<span class="sticker sticker-info">133</span>
<span class="sticker sticker-primary">133</span>
<span class="sticker sticker-success">133</span>
<span class="sticker sticker-warning">133</span>

Stickers come in 3 different sizes, small, regular and large. Create your own custom size by changing the width, height, line-height, and font-size. You can also use the sass monospace mixin to change the width and height e.g. @include monospace(30px);.

133 133 133 
<span class="sticker sticker-default sticker-sm">133</span>
<span class="sticker sticker-primary">133</span>
<span class="sticker sticker-lg sticker-success">133</span>

Change the borders with sticker-circle, sticker-rounded, or sticker-square.

133 133 133 
<span class="sticker sticker-circle sticker-default">133</span>
<span class="sticker sticker-primary sticker-rounded">133</span>
<span class="sticker sticker-square sticker-success">133</span>

Stickers are positioned in the top left corner by default. Use a combination of the classes sticker-bottom and sticker-right to position in each corner.
