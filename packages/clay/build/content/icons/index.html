Icons

Monospaced Icons

Add class icon-monospaced on an icon to make each icon occupy the same amount of horizontal space.

<span class="icon-monospaced icon-glass"></span>
<span class="icon-monospaced icon-music"></span>
<span class="icon-monospaced icon-search"></span>
<span class="icon-monospaced icon-ok"></span>
<span class="icon-monospaced icon-zoom-in"></span>
<span class="icon-monospaced icon-envelope"></span>
<span class="icon-monospaced icon-heart"></span>

Help Icon

Icon with a circular border, use for creating helpful tooltips for your users.

Icon Sizes

Add class icon-large, icon-2x, icon-3x, icon-4x, and icon-5x to change the size of an icon, increases the size by x1.33, x2, x3, x4, and x5, respectively.

<span class="icon-large icon-heart"></span>
<span class="icon-2x icon-heart"></span>
<span class="icon-3x icon-heart"></span>
<span class="icon-4x icon-heart"></span>
<span class="icon-5x icon-heart"></span>

Stacked Icons

Use <span class="icon-stack"></span> to stack an icon or text on top of another icon, some minor positional adjustments may be required.

PDF 
<span class="icon-stack">
    <span class="icon-stop" style="font-size:32px;"></span>
    <span class="icon-magnet" style="color: #FFF;"></span>
</span>

<span class="icon-stack">
    <span class="icon-comment" style="font-size:32px;"></span>
    <span class="icon-twitter" style="color: #FFF;"></span>
</span>

<span class="icon-stack">
    <span class="icon-file" style="font-size:32px;"></span>
    <span class="icon-text" style="font-size: 8px;top: 3px;">PDF</span>
</span>

Spinning Icons

Add class icon-spin on any icon to make it animate clockwise.

<span class="icon-asterisk icon-spin"></span>
<span class="icon-spinner icon-spin"></span>
<span class="icon-refresh icon-spin"></span>

Rotate and Flip

Rotate or flip an icon () with classes:

icon-rotate-90
icon-rotate-180
icon-rotate-270
icon-flip-horizontal
icon-flip-vertical
<span class="icon-rotate-90 icon-beaker"></span>
<span class="icon-rotate-180 icon-beaker"></span>
<span class="icon-rotate-270 icon-beaker"></span>
<span class="icon-flip-horizontal icon-beaker"></span>
<span class="icon-flip-vertical icon-beaker"></span>
