Progress States

Add has-danger, has-info, has-success, or has-warning to progress-group or progress to provide visual feedback for different progress states. Color a block of text or icon by wrapping it with progress-group-feedback.

Using the state classes will set the background-color on progress-bar, no need to use Bootstrap 4 background utility classes.
