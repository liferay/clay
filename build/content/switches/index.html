Switches

Toggling labels

  <input class="switch" id="switch" type="checkbox" />
  <label for="switch"><span class="on">On</span><span class="off">Off</span></label>
</div>

Edit mode

<div class="form-group">
  <input checked="checked" class="edit switch" id="switch" type="checkbox" />
  <label for="switch">Edit</label>
</div>

Disabled inputs

<div class="form-group">
  <input class="switch" disabled="disabled" id="switch" type="checkbox" />

</div>
