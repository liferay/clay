Switches

Default Switch

For customizing switches, see extending-switches.

Browser must support :checked pseudo class on checkbox element. For older browsers set css on input.switch to position: static and hide the .switch-bar element to display a regular checkbox.

<div class="form-group">
    <label>
        <input class="toggle-switch" type="checkbox">

        <span aria-hidden="true" class="switch-bar">
            <span class="switch-toggle"></span>
        </span>
    </label>
</div>

Use data attributes data-label-on="" and data-label-off="" on <span class="switch-toggle"></span> to display specific text when the switch is on and off.

<div class="form-group">
    <label>
        <input class="toggle-switch" type="checkbox">
        Switch with data-label-on

        <span aria-hidden="true" class="switch-bar">
            <span class="switch-toggle" data-label-off="" data-label-on="ON">
            </span>
        </span>
    </label>
</div>
<div class="form-group">
    <label>
        <input class="toggle-switch" type="checkbox">
        Switch with data-label on and data-label-off

        <span aria-hidden="true" class="switch-bar">
            <span class="switch-toggle" data-label-off="Switch is off." data-label-on="Switch is on.">
            </span>
        </span>
    </label>
</div>

Switches with Icons

Add an icon to the switch for the on position with <span class="button-icon button-icon-on switch-icon icon-volume-up"></span>.

Add an icon to the switch for the off position with <span class="button-icon button-icon-off switch-icon icon-volume-off"></span>.

Substitute icon-volume-up / icon-volume-down with any font awesome icon.

<div class="form-group">
    <label>
        <input class="toggle-switch" type="checkbox">
        Switch with data-label on and data-label-off

        <span aria-hidden="true" class="switch-bar">
            <span class="switch-toggle" data-label-off="Locked" data-label-on="Edit">
                <span class="button-icon button-icon-on switch-icon icon-pencil"></span>
                <span class="button-icon button-icon-off switch-icon icon-lock"></span>
            </span>
        </span>
    </label>
</div>

Switches with Icons (Alternate)

Add an icon to the switch for the on position with <span class="switch-icon-on switch-icon icon-ok"></span>.

Add an icon to the switch for the off position with <span class="switch-icon-off switch-icon icon-remove"></span>.

Substitute icon-ok / icon-remove with any font awesome icon.

<div class="form-group">
    <label>
        <input class="toggle-switch" type="checkbox">

        <span aria-hidden="true" class="switch-bar">
            <span class="switch-toggle">
                <span class="switch-icon-on switch-icon icon-ok"></span>
                <span class="switch-icon-off switch-icon icon-remove"></span>
            </span>
        </span>
    </label>
</div>

Disabled Switches

Disable a switch by adding the attribute disabled="disabled" to <input type="checkbox">.

<div class="form-group">
    <label>
        <input class="toggle-switch" disabled type="checkbox">

        <span aria-hidden="true" class="switch-bar">
            <span class="switch-toggle">
                <span class="switch-icon-on switch-icon icon-ok"></span>
                <span class="switch-icon-off switch-icon icon-remove"></span>
            </span>
        </span>
    </label>
</div>

Toggle Card

A checkbox inside a card.

<aui:input type="toggle-card" icon="icon-ok" label="Regular Label" />
<aui:input type="toggle-card" icon="icon-lock" iconOff="icon-ok" label="Regular Label" />
<aui:input type="toggle-card" icon="icon-lock" iconOff="icon-ok" label="labelOff" labelOn="labelOn" />
