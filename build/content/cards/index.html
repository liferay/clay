Horizontal Card

Display card content horizontally. .card-horizontal uses a simplified version of Bootstrap 3's grid system to help align content. It is based on the 12 column grid (default), but columns can be customized to any size.

Card columns do not break at smaller screen widths. They were designed to help align small chunks of content and are not meant to be used for page layouts.

Card-horizontal maybe omitted if not using box-shadows.

Truncating Text Inside Horizontal Card without Card Row Layout Fixed

If the text inside your container is shorter than the container, add class clamp-horizontal to card-col-{name} and wrap the content in clamp-container. If the text is larger than your container use clamp-all and wrap the content in clamp-container. Clamp-horizontal will vertically center and prevent content from overflowing horizontally. Clamp-all will prevent content from overflowing vertically and horizontally, but will not vertically center. These should generally be used when you want to keep the card size fixed, but have variable content size.

Use clamp-all-scroll instead of clamp-all to show scroll bars.
