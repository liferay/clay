## Best Pratices

<Pagination />

### Items as Anchors or Buttons

By default, all Pagination indicators were built using `buttons` elements but if you want to use as links, use [`hrefConstructor`](#api-hrefConstructor) API to construct the hyperlink references for each Pagination Indicator.

<PaginationWithLinks />

### Ellipsis Buffer

Use [`Ellipsis Buffer`](#api-ellipsisBuffer) property to define the number of pages to show on each side of the active page before using an ellipsis dropdown.

See the example below:

<PaginationEllipsisBuffer />

### Disabled Pages

Consider that you want to disable some pages on pagination, you can do that using [`disabledPages`](#api-disabledPages) property, you just need to pass an array containing the page indexes that you want to disable. See the following example:

<PaginationDisabledPages />
