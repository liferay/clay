## Best Pratices

Use [`checked`](#api-checked) property to check or not the checkbox.

<div class="clay-site-alert alert-warning">
    If you want a checkbox without actions on change(or just not use onChange property), use <code class="gatsby-code-text">readOnly</code> property.
</div>

<Checkbox />

Use [`indeterminate`](#api-indeterminate) property for making the checkbox indeterminate.

<CheckboxIndeterminate />

To set a callback function when the value of the checkbox is changed, use [`onChange`](#api-onChange) property. See the following example:

<CheckboxWithCallback />

If you want to modify the container that wraps the checkbox, use [`containerProps`](#api-containerProps):

<CheckboxContainerProps />

Use [`label`](#api-label) property to describe what the checkbox is for.

<CheckboxLabel />

Group checkboxes on the same horizontal row by using [`inline`](#api-inline) property.

<CheckboxesInline />
