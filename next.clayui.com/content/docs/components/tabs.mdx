## Composing

`ClayTabs` is a [uncontrolled component](https://reactjs.org/docs/uncontrolled-components.html), we left this component uncontrolled because the user can use Dropdowns or another control to select a `TabPane` for render.

For composing a Tab component, use `ClayTabs` component for defining a modern style, described by Lexicon, use `modern` property. If you need to justify the texts inside the `ClayTabs.Item`, use `justified` property.

Use `ClayTabs.Item` for defining a TabPane's trigger. For using this trigger as an `anchor` element, just define `href` property, otherwise, the trigger will be a `button`. Use `ClayTabs.TabPane` for wrapping the content that will be rendered by TabPane as the following example:

<Tabs />

As an uncontrolled component, Tabs allows you to control the `active` TabPane. Thereby, we can select the active TabPane by a Dropdown menu. For example:

<TabsDropdown />
