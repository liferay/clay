title: 'Inline Item'
description: ''
---

Use the `inline-item inline-item-before`, `inline-item inline-item-middle`, and `inline-item inline-item-after` classes to provide consistent spacing between text and icons in elements such as links or buttons. This can also be used in other components that have inline content (e.g., `badge` and `label`).

<div className="clay-site-alert alert alert-warning">
	For this component to function and space properly, the white space must be
	removed between text and icons from the markup inside the component. It's
	generally easier to remove all white space inside the link, button, or
	component. In the examples below, whitespaces in button and link are
	slightly off and the link underline on hover is a bit wide.
</div>

## Close

A Button or Link for closing stuff.

### Anchor

<div className="sheet-example">
	<a aria-label="Close" className="close" href="#1" role="button">
		<svg
			className="lexicon-icon lexicon-icon-times"
			focusable="false"
			role="presentation"
		>
			<use href="/images/icons/icons.svg#times" />
		</svg>
	</a>
	<a
		aria-label="Close"
		className="close disabled"
		href="#1"
		role="button"
		tabindex="-1"
	>
		<svg
			className="lexicon-icon lexicon-icon-times"

		className="close disabled"
		href="#1"
		role="button"

	>
		<span aria-hidden="true">×</span>
	</a>
</div>
