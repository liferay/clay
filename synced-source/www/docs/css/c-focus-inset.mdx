---
title: 'C Focus Inset'
description: ''
---

A bailout utility to set the focus border inside the element, just add the class `c-focus-inset`.

## Examples

### Button

<button className="btn btn-primary c-focus-inset" type="button">
	Primary
</button>
<button className="btn btn-secondary c-focus-inset" type="button">
	Secondary
</button>
<button className="btn btn-info c-focus-inset" type="button">
	Info
</button>
<button className="btn btn-success c-focus-inset" type="button">
	Success
</button>
<button className="btn btn-warning c-focus-inset" type="button">
	Warning
</button>
<button className="btn btn-danger c-focus-inset" type="button">
	Danger
</button>
<button className="btn btn-light c-focus-inset" type="button">
	Light
</button>
<button className="btn btn-dark c-focus-inset" type="button">
	Dark
</button>
<button className="btn btn-link c-focus-inset" type="button">
	Link
</button>
<button className="btn btn-unstyled c-focus-inset" type="button">
	Unstyled
</button>

```html
<button className="btn btn-primary c-focus-inset" type="button">Primary</button>
```

### Link

<div>
	<a className="c-focus-inset" href="javascript:;">
		Regular Anchor Tag
	</a>
</div>
<div>
	<a className="c-focus-inset link-primary single-link" href="javascript:;">
		.link-primary.single-link
	</a>
</div>
<h4 className="component-title">
	<a className="c-focus-inset" href="javascript:;">
		.component-title a
	</a>
</h4>
<a className="component-title c-focus-inset" href="javascript:;">
	a.component-title
</a>
<p className="component-subtitle">
	<a className="c-focus-inset" href="javascript:;">
		.component-subtitle a
	</a>
</p>
<a className="component-subtitle c-focus-inset" href="javascript:;">
	a.component-subtitle
</a>

```html
<a className="c-focus-inset" href="javascript:;">Regular Anchor Tag</a>
```
