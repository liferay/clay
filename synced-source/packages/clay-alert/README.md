# `@clayui/alert`

Alerts are used to capture the attention of the user in an intrusive way.

-   [Documentation](https://clayui.com/docs/components/alert.html)
-   [Changelog](./CHANGELOG.md)

## Install

Run `yarn`

```shell
yarn add @clayui/alert
```
