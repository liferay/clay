## Structure

Sidebar is an opinionated container to display related content.

Is composed by:

```html
<div className="sidebar-header">Sidebar Header</div>
```

```html
<div className="sidebar-body">Sidebar Body</div>
```

```html
<div className="sidebar-footer">Sidebar Footer</div>
```
