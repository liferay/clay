---
title: 'Breadcrumb'
description: 'Breadcrumb is a secondary navigation pattern that identifies the page position inside a hierarchy.'
lexiconDefinition: 'https://liferay.design/lexicon/core-components/navigation/breadcrumb/'
packageNpm: '@clayui/breadcrumb'
packageUse: "import Breadcrumb from '@clayui/breadcrumb';"
---

A navigation aid for your site, provide a quick way to jump back to previously viewed pages or sections.

Use `<span className="breadcrumb-text-truncate"></span>` inside breadcrumb links to truncate text based on a max-width.

<div className="clay-site-alert alert alert-warning">
	Breadcrumb will use{' '}
	<code className="gatsby-code-text">$breadcrumb-divider-svg-icon</code> by
	default, set $breadcrumb-divider-svg-icon: none; if you wish to use the
	UTF-8 charset or third-party icon font.
</div>

<div className="clay-site-alert alert alert-warning">
	Don't forget to check{' '}
	<a href="https://www.w3.org/TR/wai-aria-practices/#breadcrumb">WAI-ARIA</a>{' '}
	accessibility pratices for alerts when writting your markup.
</div>
