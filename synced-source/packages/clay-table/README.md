# `@clayui/table`

A table is a specific pattern for comparing datasets in a very direct and analytical way.

-   [Documentation](https://clayui.com/docs/components/table.html)
-   [Changelog](./CHANGELOG.md)

## Install

Run `yarn`

```shell
yarn add @clayui/table
```
