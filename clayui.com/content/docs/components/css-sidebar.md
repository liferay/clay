## Structure

Sidebar is an opinionated container to display related content.

Is composed by:

```html
<div class="sidebar-header">
	Sidebar Header
</div>
```

```html
<div class="sidebar-body">
	Sidebar Body
</div>
```

```html
<div class="sidebar-footer">
	Sidebar Footer
</div>
```
